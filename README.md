# RS_Dashboard

Ferramenta de avaliação da composição corporal e planejador de gasto energético

## About

RS_Dashboard é um projeto desenvolvido para avaliar a composição corporal e planejar o gasto energético de forma prática e acessível.

## Features

- Avaliação de composição corporal
- Planejador de gasto energético
- Interface intuitiva e responsiva
